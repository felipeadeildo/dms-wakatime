# dms-wakatime

WakaTime plugin for [DankMaterialShell](https://danklinux.com).

Monitor your coding time directly from the bar. Supports WakaTime and
self-hosted alternatives like Wakapi and Hakatime.


## Requirements

- `curl`
- WakaTime CLI configured (`~/.wakatime.cfg`)

## Installation

```bash
git clone https://github.com/felipeadeildo/dms-wakatime ~/.config/DankMaterialShell/plugins 
dms restart
```

Then: Settings -> Plugins -> Scan -> Enable WakaTime
